# Distance in the Shadows

![Distance in the Shadows](https://distanceintheshadows.vercel.app/img/hero.png)

**Distance in the Shadows** is a helper extension for running Shadowdark RPG on Owlbear Rodeo. It draws native, performance-friendly vector semi-circles indicating Close (5ft), Near (up to 30ft), and Far (+30ft) boundaries directly attached to character, mount, or prop tokens.

## Features

- **Context Menu Toggle**: Right-click character, mount, or prop tokens and select **SD Distances** to toggle the measurement template on or off.
- **Proportional Token Scaling**: Distances scale automatically based on the token's size, so large creatures automatically project larger reach templates.
- **Synchronized Rotation**: Drag the standard rotate handle to aim the semi-circle. The Near and Far areas are nested attachments and rotate with it automatically.
- **5 Custom Color Palettes**: Choose from Classic Gray, Royal Violet, Emerald Forest, Ocean Blue, or Fiery Amber — active templates on the map update instantly.
- **Clear Scene Utility**: Quickly clean up all active templates on the map with a single click.

## Distance Bands

| Band | Range | Description |
| :--- | :--- | :--- |
| **Close** | 5 ft | Directly adjacent — inner semi-circle |
| **Near** | Up to 30 ft | Up to 6 grid cells outward |
| **Far** | 30+ ft boundary | Thin 5ft boundary layer marking the start of Far |
